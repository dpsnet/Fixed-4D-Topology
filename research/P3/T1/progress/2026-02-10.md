### 2. 重大发现

**凸性分析结果**:
```
E(d) 严格凸: ✓ 当 α+β > 0
S(d) 严格凹: ✓ 恒成立
F(d) = E(d) - T·S(d): 有条件凸
```

**非凸区域发现**:
- 当 α, β 很小且 T 很大时，F(d) 不是凸的
- 例子: α=0.1, β=0.1, T=1.0 → F''_min = -0.10 < 0

**凸性条件**:
```
F''(d) = 2(α+β) + T/d² > 0

最严格条件在 d=2:
2(α+β) + T/4 > 0

或者:
α + β > T/8
```

## 数学结论

**定理** (凸性充分条件):
如果 α + β > T/8，则总泛函 F(d) 在 [2,4] 上严格凸。

**证明**:
```
F''(d) = 2(α+β) - T/d² ≥ 2(α+β) - T/4  (因为 d ≤ 4)

要求 F''(d) > 0:
2(α+β) - T/4 > 0
=> α + β > T/8
```
