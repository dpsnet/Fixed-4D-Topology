## 数学框架

### 1. 谱维度回顾

谱维度定义为热核迹的渐近行为：
```
d_s = -2 lim_{t→∞} ln Z(t) / ln t
Z(t) = Tr(e^{-tΔ})
```

### 2. 示性类基础

**Chern类** (复向量丛):
```
c(E) = det(I + iF/2π) = Σ c_k(E)
```

**Pontryagin类** (实向量丛):
```
p(E) = det(I + F/2π) = Σ (-1)^k p_k(E)
```

### 3. 指标定理

**Atiyah-Singer指标定理**:
```
index(D) = ∫_M Â(TM) ∧ ch(E)
```
