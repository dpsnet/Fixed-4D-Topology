## 深入分析

### 流向分析的关键

Master方程: dd/d(ln μ) = β(d)

去向某个极限时：
- dd = β × d(ln μ)

**去向UV (μ→∞)**:
- ln μ → +∞, d(ln μ) > 0
- 如果 β < 0, 则 dd < 0, d减小 → 流向d=2

**去向IR (μ→0)**:
- ln μ → -∞, d(ln μ) < 0
- 如果 β < 0, 则 dd > 0, d增加 → 流向d=4

**标准模型**: β = -α(d-2)(4-d)

在2<d<4区间:
- (d-2) > 0
- (4-d) > 0
