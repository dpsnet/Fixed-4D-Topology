## 1. 加法的定义

### 1.1 问题重述

给定两个分形维数$d_1, d_2 \in \mathcal{D}$，定义它们的"加法"$d_1 \oplus d_2$。

**候选定义**：

**D1**: 实数加法
$$d_1 \oplus d_2 = d_1 + d_2$$

**D2**: 调和型
$$d_1 \oplus d_2 = \frac{2d_1 d_2}{d_1 + d_2}$$

**D3**: 几何平均
$$d_1 \oplus d_2 = \sqrt{d_1 d_2}$$

**D4**: 分形笛卡尔积
$$d_1 \oplus d_2 = \dim(F_1 \times F_2) = d_1 + d_2$$

---

## 2. 笛卡尔积加法（采纳）

### 2.1 几何动机

**分形的笛卡尔积**：

若$F_1, F_2$是分形，则$F_1 \times F_2$也是分形（适当定义下）。

**维数公式**：

$$\dim_H(F_1 \times F_2) = \dim_H(F_1) + \dim_H(F_2) = d_1 + d_2$$

**证明**：对于自相似分形，乘积的构造参数是原参数的乘积，维数相加。

### 2.2 严格定义

**定义2.1 (分形加法)**

对于$d_1, d_2 \in \mathcal{D}$：

$$d_1 \oplus d_2 = d_1 + d_2$$

**解释**：这是分形笛卡尔积的维数。

---

## 3. 代数性质分析

### 3.1 封闭性

**问题**：$d_1 + d_2$是否仍是分形维数？

**反例**：

设$d_1 = \frac{\log 2}{\log 3} \approx 0.631$（Cantor集）
$d_2 = \frac{\log 2}{\log 5} \approx 0.431$（Cantor五分集）

$$d_1 + d_2 = \frac{\log 2}{\log 3} + \frac{\log 2}{\log 5} = \log 2 \cdot \left(\frac{1}{\log 3} + \frac{1}{\log 5}\right)$$

$$= \frac{\log 2 \cdot (\log 3 + \log 5)}{\log 3 \cdot \log 5} = \frac{\log 2 \cdot \log 15}{\log 3 \cdot \log 5}$$

**问题**：能否写成$\frac{\log N}{\log (1/r)}$的形式？

一般不能！这需要：

$$\frac{\log 2 \cdot \log 15}{\log 3 \cdot \log 5} = \frac{\log N}{\log (1/r)}$$

解这个方程需要特定的$N$和$r$，一般不存在。

**结论**：分形维数在加法下**不封闭**。

---

### 3.2 闭包构造

**定义3.1 (分形维数加法闭包)**

$$\bar{\mathcal{D}} = \left\{\sum_{i=1}^k d_i : k \in \mathbb{N}, d_i \in \mathcal{D}\right\}$$

即分形维数在加法下的生成集合。

**性质**：
- $\bar{\mathcal{D}} \supset \mathcal{D}$
- $\bar{\mathcal{D}}$在加法下封闭
- $\bar{\mathcal{D}} \subset \mathbb{R}^+$（正实数）

---

### 3.3 半群结构

**定理3.2 (加法半群)**

$(\bar{\mathcal{D}}, \oplus)$是**交换半群**。

**证明**：

1. **封闭性**：由构造，$\bar{\mathcal{D}}$在$\oplus$下封闭
2. **结合律**：$(d_1 \oplus d_2) \oplus d_3 = (d_1 + d_2) + d_3 = d_1 + (d_2 + d_3) = d_1 \oplus (d_2 \oplus d_3)$
3. **交换律**：$d_1 \oplus d_2 = d_1 + d_2 = d_2 + d_1 = d_2 \oplus d_1$

**证毕**。

---

### 3.4 单位元与逆元

**单位元**：

需要$e$使得$d \oplus e = d$，即$d + e = d$，所以$e = 0$。

**问题**：$0 \in \bar{\mathcal{D}}$吗？

**回答**：不在！分形维数$> 0$，有限和也$> 0$。

**结论**：$(\bar{\mathcal{D}}, \oplus)$没有单位元，是**纯半群**（无单位元的半群）。

**扩展**：

定义$\bar{\mathcal{D}}_0 = \bar{\mathcal{D}} \cup \{0\}$，则$(\bar{\mathcal{D}}_0, \oplus)$有单位元$0$。

**逆元**：

对于$d > 0$，$-d < 0 \notin \bar{\mathcal{D}}_0$。

**结论**：没有逆元，不是群。

## 4. 序结构

### 4.1 全序

$(\bar{\mathcal{D}}, \leq)$是全序集（继承$\mathbb{R}$的序）。

### 4.2 单调性

加法关于序是单调的：

$$d_1 \leq d_2 \implies d_1 \oplus d_3 \leq d_2 \oplus d_3$$

### 4.3 Archimedean性质

**验证**：对于任意$d_1, d_2 \in \bar{\mathcal{D}}$，$d_1 > 0$，存在$n \in \mathbb{N}$使得：

$$n \cdot d_1 \geq d_2$$

**成立**：因为$\bar{\mathcal{D}} \subset \mathbb{R}$，实数具有Archimedean性质。

T1证明了$\mathcal{V}_C = \text{span}_{\mathbb{Q}}(\mathcal{D}_C)$在$\mathbb{R}$中稠密。

- $\bar{\mathcal{D}}$：有限和（整数系数）

## 6. 特殊子结构

### 6.1 固定压缩比的子半群

回顾A1的发现：

**定理6.1 (固定压缩比的封闭性)**

对于固定压缩比$r$，集合：

$$\mathcal{D}^{(r)} = \left\{\frac{\log N}{\log (1/r)} : N \in \mathbb{N}, N \geq 2\right\}$$

在加法下封闭。

**证明**：

$$\frac{\log N_1}{\log (1/r)} + \frac{\log N_2}{\log (1/r)} = \frac{\log (N_1 N_2)}{\log (1/r)} \in \mathcal{D}^{(r)}$$

**证毕**。

**推论**：$(\mathcal{D}^{(r)}, \oplus)$是**交换半群**。

**进一步**：

$(\mathcal{D}^{(r)}, \oplus) \cong (\mathbb{N}_{\geq 2}, \times)$

通过同构：$\frac{\log N}{\log (1/r)} \mapsto N$

**结构识别**：$(\mathcal{D}^{(r)}, \oplus)$同构于正整数乘法半群。
