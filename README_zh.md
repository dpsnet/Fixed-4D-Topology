## 核心定理

1. **主方程**: d_eff = argmin[E - T·S + Λ]
2. **谱公式**: d_s(t) = n - (R/3)t + O(t²)
3. **凸性**: α + β > T/8
4. **Cantor 复杂度**: C* = (Δλ/λ₁) · d_c · (1-d_c) · π/4
5. **统一权重**: w_i ∝ 1/|λ_i| at criticality
6. **幺正等价**: H_NN = U · L_network · U†
