## 核心定理

1. **Cantor逼近**: 贪婪算法收敛率 O(3^-n)
2. **Master方程**: d_eff = argmin[E - T·S + Λ]
3. **谱公式**: d_s(t) = n - (R/3)t + O(t²)
4. **凸性**: F(d) 严格凸 ⟺ α + β > T/8
