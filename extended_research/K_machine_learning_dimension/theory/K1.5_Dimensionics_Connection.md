# K1.5: 与Dimensionics框架的连接

## 1. 统一愿景

将K方向（机器学习维度）纳入Dimensionics-Physics的统一框架，建立跨尺度的维度理论。

## 2. 核心对应关系

### 2.1 概念映射

| Dimensionics-Physics | 机器学习 (K方向) | 对应关系 |
|---------------------|-----------------|---------|
| 能量尺度 $\mu$ | 训练时间 $t$ | 单调递增 |
| 谱维度 $d_s(\mu)$ | 有效维度 $d_{\text{eff}}(t)$ | 直接对应 |
| Master Equation | SGD Dynamics | 形式类比 |
| UV固定点 ($\mu \to \infty$) | 早训练期 ($t \to 0$) | 初始状态 |
| IR固定点 ($\mu \to 0$) | 收敛期 ($t \to \infty$) | 稳态 |
| Beta函数 $\beta(d)$ | 维度演化函数 $f(d, \mathcal{L})$ | 功能等价 |

### 2.2 统一主方程

**Dimensionics主方程**:
$$\mu \frac{\partial d_s}{\partial \mu} = \beta(d_s)$$

**K方向主方程** (时间版本):
$$t \frac{\partial d_{\text{eff}}}{\partial t} = \beta_{\text{NN}}(d_{\text{eff}}; \mathcal{L})$$

其中:
$$\beta_{\text{NN}}(d; \mathcal{L}) = \alpha \mathcal{L}(d_{\text{data}} - d) - \gamma d R$$

### 2.3 尺度变换

能量-时间对偶:
$$\mu \sim \frac{1}{t}$$

这反映了:
- 高能量 ↔ 早训练（快速变化）
- 低能量 ↔ 晚训练（缓慢收敛）

## 3. Beta函数的ML形式

### 3.1 显式表达式

$$\beta_{\text{NN}}(d) = -\alpha_{\text{NN}} (d - d_{\text{UV}})(d - d_{\text{IR}})$$

其中:
- $d_{\text{UV}} = d_{\text{eff}}^{\text{init}}$: 初始维度（高复杂度）
- $d_{\text{IR}} = d_{\text{data}}$: 数据维度（目标复杂度）
- $\alpha_{\text{NN}}$: 学习率相关的系数

### 3.2 解析解

与Dimensionics类似，维度演化有解析解:

$$d_{\text{eff}}(t) = d_{\text{IR}} + \frac{d_{\text{UV}} - d_{\text{IR}}}{1 + \left(\frac{t}{t_c}\right)^{\alpha_{\text{NN}}}}$$

其中 $t_c$ 是特征时间尺度。

### 3.3 固定点分析

**UV固定点** ($t \to 0$):
$$\lim_{t \to 0} d_{\text{eff}}(t) = d_{\text{UV}}$$

对应随机初始化的高维状态。

**IR固定点** ($t \to \infty$):
$$\lim_{t \to \infty} d_{\text{eff}}(t) = d_{\text{IR}} = d_{\text{data}}$$

对应学习数据结构后的稳态。
