# 行为准则

## 我们的承诺

作为本项目的贡献者和维护者，我们致力于营造一个开放、友好的环境，无论：

- 经验水平
- 教育背景
- 身份特征
- 个人背景

## 我们的标准

### 积极行为

- 使用友好、包容的语言
- 尊重不同观点和经验
- 接受建设性批评
- 关注社区利益
- 帮助新手入门

### 不可接受行为

- 歧视性语言或行为
- 人身攻击或侮辱
- 骚扰或恐吓
- 未经同意的个人信息发布
- 其他不专业行为

## 责任

项目维护者有责任：
- 明确行为标准
- 处理不当行为报告
- 采取适当纠正措施

## 范围

本行为准则适用于：
- 项目代码仓库
- Issue讨论
- Pull Request
- 社区活动

## 执行

如发现不当行为，请通过GitHub Issue报告。我们将：
- 及时调查
- 保护报告者隐私
- 采取必要措施

## 联系

举报不当行为：
- GitHub Issues: https://github.com/dpsnet/Fixed-4D-Topology/issues
