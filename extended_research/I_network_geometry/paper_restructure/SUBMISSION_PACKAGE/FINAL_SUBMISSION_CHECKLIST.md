## 阶段2: 投稿材料最终检查 (Day 3)

### 文件检查

**核心文件** (必须上传)
- [ ] 主论文 (Main Manuscript)
  - 格式: .tex/.docx/.pdf
  - 文件名: Manuscript.{ext}
  
- [ ] 封面信 (Cover Letter)
  - 格式: .pdf/.docx
  - 文件名: Cover_Letter.{ext}

- [ ] 亮点 (Highlights)
  - 通常在线填写或单独文件
  
**补充文件** (必需)
- [ ] 数据可用性声明
- [ ] 作者贡献声明
- [ ] 利益冲突声明
- [ ] 资助声明

**补充材料** (如需要)
- [ ] 补充信息 (Supplementary Information)
- [ ] 补充图表
- [ ] 代码仓库链接

### 内容检查

**摘要**
- [ ] 长度符合要求 (通常150-250 words)
- [ ] 概括研究背景、方法、结果、意义
- [ ] 无引用

**亮点**
- [ ] 3-5条
- [ ] 每条长度符合要求
- [ ] 突出核心发现

**图表**
- [ ] 分辨率足够 (300 dpi)
- [ ] 字体清晰
- [ ] 图例完整
- [ ] 编号正确

**参考文献**
- [ ] 格式符合期刊要求
- [ ] 所有引用都有对应文献
- [ ]  BioGRID正确引用 (Stark et al. 2006)
- [ ] SNAP数据集正确引用

## 阶段3: 在线投稿 (Day 4)

### 投稿系统操作

**步骤1: 开始投稿**
- [ ] 登录投稿系统
- [ ] 点击"Submit New Manuscript"
- [ ] 选择文章类型 (Research Article)

**步骤2: 上传文件**
- [ ] 上传主论文
- [ ] 上传封面信
- [ ] 上传补充材料
- [ ] 确认文件分类正确

**步骤3: 填写元数据**
- [ ] 标题
- [ ] 作者信息 (姓名、单位、邮箱)
- [ ] 摘要 (粘贴或上传)
- [ ] 关键词 (3-6个)
  - 建议: network dimension, complex networks, box-counting, 
    empirical study, biological networks, infrastructure networks

**步骤4: 推荐审稿人**
- [ ] 推荐3-5位审稿人
- [ ] 提供姓名、单位、邮箱
- [ ] 简要说明推荐理由
- [ ] 避免利益冲突人员

**步骤5: 回答问题**
- [ ] 是否 previously published? (No)
- [ ] 是否同时投稿其他期刊? (No)
- [ ] 确认所有作者同意投稿
- [ ] 确认数据可用性

**步骤6: 最终检查**
- [ ] 预览PDF
- [ ] 检查所有信息正确
- [ ] 确认无遗漏文件

**步骤7: 提交**
- [ ] 点击"Submit"
- [ ] 保存投稿编号
- [ ] 截图确认页面
- [ ] 查收确认邮件

## 阶段4: 投稿后跟进

### 即时确认 (Day 4)

- [ ] 收到自动确认邮件
- [ ] 记录投稿编号
- [ ] 保存所有相关文件

### 审稿周期 (Week 2-8)

**Week 2-4: 初审**
- [ ] 等待编辑初审
- [ ] 可能被desk reject (快速拒稿)
- [ ] 或进入审稿流程

**Week 4-8: 外审**
- [ ] 等待审稿人意见
- [ ] 通常2-3位审稿人
- [ ] 审稿周期因期刊而异

### 可能结果

**Accept** (罕见)
- [ ] 直接接受
- [ ] 可能需要小修改

**Minor Revision** (小修)
- [ ] 1-2周修改时间
- [ ] 回复审稿意见
- [ ] 重新提交

**Major Revision** (大修)
- [ ] 1-3个月修改时间
- [ ] 需要补充实验/分析
- [ ] 大幅修改后重新提交

**Reject** (拒稿)
- [ ] 查看拒稿原因
- [ ] 考虑转投其他期刊
- [ ] 或大幅修改后重投
