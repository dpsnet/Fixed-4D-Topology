# Data Availability Statement

## Datasets Used in This Study

All network datasets analyzed in this study are publicly available from reputable sources:

### 1. Internet AS Topology
- **Source**: CAIDA (Center for Applied Internet Data Analysis)
- **Dataset**: AS-Skitter
- **URL**: http://www.caida.org/tools/measurement/skitter/
- **Citation**: Leskovec J, Kleinberg J, Faloutsos C. Graphs over Time: Densification Laws, Shrinking Diameters and Possible Explanations. ACM SIGKDD 2005.
- **License**: Public academic use

### 2. DBLP Academic Collaboration Network
- **Source**: SNAP Stanford
- **Dataset**: DBLP Computer Science Bibliography
- **URL**: https://snap.stanford.edu/data/com-DBLP.html
- **Citation**: Yang J, Leskovec J. Defining and Evaluating Network Communities based on Ground-truth. ICDM 2012.
- **License**: Public academic use

### 3. Yeast Protein-Protein Interaction Network
- **Source**: BioGRID (Biological General Repository for Interaction Datasets)
- **Dataset**: Saccharomyces cerevisiae S288c (Release 5.0.254)
- **URL**: https://downloads.thebiogrid.org/
- **Citation**: Stark C, Breitkreutz BJ, Reguly T, Boucher L, Breitkreutz A, Tyers M. BioGRID: A General Repository for Interaction Datasets. Nucleic Acids Res. 2006; 34:D535-9.
- **License**: MIT License (100% freely available to academic and commercial users)

### 4. Facebook Social Network
- **Source**: SNAP Stanford
- **Dataset**: Facebook Combined (ego networks)
- **URL**: https://snap.stanford.edu/data/egonets-Facebook.html
- **Citation**: McAuley JJ, Leskovec J. Learning to Discover Social Circles in Ego Networks. NIPS 2012.
- **License**: Public academic use

### 5. Twitter Social Network
- **Source**: SNAP Stanford
- **Dataset**: Twitter Combined (ego networks)
- **URL**: https://snap.stanford.edu/data/egonets-Twitter.html
- **Citation**: McAuley JJ, Leskovec J. Learning to Discover Social Circles in Ego Networks. NIPS 2012.
- **License**: Public academic use

### 6. IEEE Power Grid
- **Source**: IEEE / University of Washington
- **Dataset**: IEEE 118 Bus Test Case
- **URL**: https://www2.ee.washington.edu/research/pstca/
- **Citation**: IEEE 118 Bus Test Case. Power Systems Test Case Archive. University of Washington, Electrical Engineering.
- **License**: Public academic use (standard test case)

### 7. Email Communication Network
- **Source**: SNAP Stanford
- **Dataset**: email-Eu-core
- **URL**: https://snap.stanford.edu/data/email-Eu-core.html
- **Citation**: Leskovec J, Kleinberg J, Faloutsos C. Graph Evolution: Densification and Shrinking Diameters. ACM TKDD 2007.
- **License**: Public academic use

## Data Accessibility

All datasets can be directly downloaded from the sources listed above. No restrictions apply to the use of these datasets for academic research. BioGRID data is released under the MIT License, allowing both academic and commercial use with appropriate citation.
