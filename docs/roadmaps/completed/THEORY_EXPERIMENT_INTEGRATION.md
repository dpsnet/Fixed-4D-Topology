## 核心对应关系

### 1. 实验现象 → 数学量对应

| 实验可观测量 | 数学量 | 对应关系 |
|-------------|--------|---------|
| 旋转角速度 $\omega$ | 扩散时间 $t$ | $t \sim 1/\omega^2$ |
| 有效维度 $d_{\text{eff}}$ | 谱维 $d_s(t)$ | $d_{\text{eff}} = d_s(t) + 1$ (含时间维) |
| 小球分布范围 | 热核迹 $\Theta(t)$ | 通过盒计数法关联 |
| 不同质量小球 | 不同分形结构 | 质量 ↔ Hausdorff维数 $

### 2. 实验观察到的谱维流动

**E-6实验观察**:
```
转速:     0rpm → 200rpm → 500rpm → 1000rpm
能量:     低   →   中   →   高   →  极高
空间维:   3D  →   ~2.9D →  ~2.6D →  ~2.2D
谱维:     ~4  →   ~3.9  →   ~3.6  →   ~3.2
          (含时间维)
```

**与Theorem A的联系**:

Theorem A给出热核迹渐近:
$$\Theta_\Gamma(t) = \frac{\text{Vol}}{(4\pi t)^{3/2}} + c(\delta) t^{-(1+\delta)/2} + O(t^{-1/2})$$

谱维定义:
$$d_s(t) = -2 \frac{d\ln \Theta(t)}{d\ln t}$$

**计算谱维流动**:

对于Theorem A的渐近行为:
- 大 $t$ (低能/低转速): 主项主导, $d_s \approx 3$ (空间) + 1 = 4
- 小 $t$ (高能/高转速): 分形项主导, $d_s \approx (1+\delta)/2 \times 2 = 1+\delta \approx 2.2$ (当 $\delta \approx 1.2$)

这与E-6实验观察到的 $4 \to 3.2$ 流动一致！
