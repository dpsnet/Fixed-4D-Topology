## Section 3: Pre-Send Checklist

### For Each Expert, Before Sending Email:

#### Document Preparation
- [ ] Project Overview PDF generated
- [ ] Technical questions relevant to expert extracted
- [ ] Proof summary sections selected appropriately
- [ ] All PDFs combined into single attachment (optional)
- [ ] File sizes checked (< 10 MB total recommended)

#### Email Content
- [ ] Subject line customized
- [ ] Opening paragraph personalized
- [ ] Three specific questions selected for this expert
- [ ] Mutual connections mentioned (if applicable)
- [ ] Tone appropriate for expert's seniority
- [ ] No spelling/grammar errors
- [ ] All hyperlinks work (if any)

#### Technical Review
- [ ] Mathematical claims in email verified
- [ ] References to expert's work accurate
- [ ] No confidential information revealed
- [ ] Attribution correct for all cited results

#### Administrative
- [ ] Reply-to address set correctly
- [ ] Email signature complete
- [ ] Attachments actually attached (double-check!)
- [ ] CC/BCC appropriate
- [ ] Send time appropriate (avoid weekends/holidays)

## Section 6: Consultation Preparation

### Before Each Consultation

#### 24 Hours Before
- [ ] Send calendar invite with video link
- [ ] Test technology (audio, video, screen sharing)
- [ ] Prepare specific questions for this expert
- [ ] Review expert's recent work
- [ ] Print/queue key documents for reference

#### Day Of
- [ ] Join call 5 minutes early
