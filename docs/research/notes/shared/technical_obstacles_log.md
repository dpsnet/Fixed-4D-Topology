## 障碍解决策略库

### 策略模板

当遇到以下类型问题时，可参考对应策略：

#### 类型1: 理论推导障碍

**识别特征**:
- 推导无法继续
- 出现矛盾结果
- 无法验证中间步骤

**解决策略**:
1. 回归基本定义
2. 简化问题场景
3. 寻找反例验证
4. 咨询领域专家
5. 文献深度调研

#### 类型2: 数值验证失败

**识别特征**:
- 数值结果与理论不符
- 计算不收敛
- 精度不达标

**解决策略**:
1. 检查算法实现
2. 增加数值精度
3. 简化测试案例
4. 交叉验证方法
5. 检查边界条件

#### 类型3: 文献缺口

**识别特征**:
- 找不到相关定理
- 现有方法不适用
- 需要全新技术

**解决策略**:
1. 扩展搜索范围
2. 寻找相关领域
3. 联系原作者
4. 数学社区求助
5. 自主开发新方法
