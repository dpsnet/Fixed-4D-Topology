### 3.2 体积计算结果

**观察**: SnapPy返回的体积值:
- d=1: 2.029883... (这与文献值0.915965不一致)
- d=2: -2.029883... (负值需要解释)
- d=3: 2.029883...

**分析**: 
- SnapPy返回的是manifold的体积，不是orbifold体积
- Bianchi orbifold的体积公式: vol = |D|^{3/2}/(4π²) · ζ_K(2)
- 对于d=1 (D=-4): 理论值约为 0.915965
- 对于d=3 (D=-3): 理论值约为 0.845785

**结论**: SnapPy的manifold体积需要与orbifold体积区分。后续计算应使用理论公式验证。

## 5. 数学洞察

### 5.1 维数趋势分析

Hausdorff维数 δ(d) 随d的增加呈现上升趋势:

```
d:    1     2     3     7     11
δ:  1.72  1.79  1.70  1.83  1.90
```

**观察**: d=3是最小值点，这与以下因素有关:
- Q(√-3)的类数为1
- 具有额外的6阶单位根
- 更大的对称性导致更小的极限集维数

### 5.2 与L-函数的关系假设

**核心公式假设**:

Hausdorff维数 δ 与Dedekind zeta函数的关系可能形如:

```
δ = 2 - c / ζ_K(2) + O(1/ζ_K(2)²)
```

或者通过谱间隙联系:
```
λ_1 = δ(2-δ)
```

其中 λ_1 是Laplace算子的第一非零特征值。

### 5.3 几何解释

- **d=1** (高斯整数): 极限集具有四重对称性
- **d=3** (Eisenstein整数): 极限集具有六重对称性，维数最小
- **大d值**: 随着d增大，群结构变得更"复杂"，极限集更"分散"，维数趋近于2
