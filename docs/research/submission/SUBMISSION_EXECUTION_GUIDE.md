在登录投稿系统之前，请确保已准备好以下材料：

## 4. 常见问题处理 (FAQ and Troubleshooting)

### 4.1 技术问题

#### Q1: 文件上传失败
**可能原因:**
- 文件大小超过限制
- 文件名包含特殊字符
- 网络连接问题

**解决方案:**
- 检查文件大小，必要时压缩
- 使用简单文件名 (仅字母、数字、下划线)
- 尝试重新登录或更换浏览器

#### Q2: LaTeX编译错误
**可能原因:**
- 缺少必要的宏包
- 类文件未上传
- 图片路径问题

**解决方案:**
- 确保所有依赖文件已上传
- 使用相对路径引用图片
- 在本地测试编译通过后再上传

#### Q3: PDF预览格式错乱
**可能原因:**
- 系统使用的编译器版本不同
- 字体问题

**解决方案:**
- 使用标准的LaTeX宏包
- 嵌入所有字体
- 接受轻微的格式差异

### 4.2 内容问题

#### Q4: 摘要长度超限
**要求:** ≤200词
**解决方案:**
- 精简背景介绍
- 聚焦主要结果
- 删除次要细节

#### Q5: 作者顺序错误
**注意:** 投稿后更改作者顺序需要编辑批准
**解决方案:**
- 投稿前仔细确认
- 如需更改，联系编辑部说明原因

#### Q6: MSC分类不确定
**解决方案:**
- 参考MathSciNet的MSC2020分类
- 咨询领域专家
- 选择最相关的主要和次要分类

### 4.3 系统问题

#### Q7: 无法登录系统
**解决方案:**
- 检查用户名和密码
- 使用密码重置功能
- 联系技术支持: scholarone@math.princeton.edu

#### Q8: 系统超时
**解决方案:**
- 及时保存进度
- 分步完成投稿
- 避免长时间离开页面

## 5. 投稿后跟进流程 (Post-Submission Follow-up)

### 5.1 立即行动 (提交后1-3天)

- [ ] 保存确认邮件和稿件编号
- [ ] 通知所有合作作者
- [ ] 备份所有提交文件
- [ ] 记录投稿日期和时间

### 5.2 短期跟进 (提交后1-4周)

- [ ] 检查邮件 (包括垃圾邮件文件夹)
- [ ] 确认编辑部收到所有材料
- [ ] 如有问题，及时响应编辑部询问
- [ ] 准备可能的格式修改

### 5.3 中期等待 (提交后1-6个月)

- [ ] 定期检查投稿系统状态
- 可能的状态变化:
  - **Submitted** → **Under Review** (找到审稿人)
  - **Under Review** → **Required Reviews Complete** (审稿完成)
  - **Decision Pending** (等待决定)

### 5.4 决定阶段 (6-12个月)

#### 可能的决定

| 决定 | 含义 | 后续行动 |
|------|------|----------|
| **Accept** | 直接接受 | 准备最终版本 |
| **Minor Revision** | 小修 | 按要求修改，1-2周返回 |
| **Major Revision** | 大修 | 按要求修改，1-3个月返回 |
| **Reject & Resubmit** | 拒稿但鼓励重投 | 重大修改后可重投 |
| **Reject** | 拒稿 | 考虑其他期刊 |

### 5.5 修订提交 (如适用)

#### 修订信撰写要点

1. **逐条回复**所有审稿意见
2. **明确标注**修改位置 (行号/页码)
3. **解释说明**未采纳的建议
4. **保持礼貌**和专业

#### 修订格式

```
Reviewer Comment: [原文]

Response: [回复]

Changes: [修改说明，如: Section 3.2, page 15, line 20-25]
[高亮修改内容]
```
