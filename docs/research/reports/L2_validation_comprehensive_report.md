## 2. 方法论

### 2.1 Kleinian群计算

#### 2.1.1 算法描述

**核心算法**: Hausdorff维数估计的盒计数法

```
算法: BoxCountingDimension
输入: 点集 P ⊂ ℝ³, 最小盒子尺寸 ε_min, 最大盒子尺寸 ε_max
输出: Hausdorff维数估计 d_H

1. 生成几何序列: ε_i = ε_max × (ε_min/ε_max)^(i/n), i = 0,...,n
2. 对每个 ε_i:
   a. 将空间划分为边长为 ε_i 的立方体网格
   b. 统计包含至少一个P中点的盒子数 N(ε_i)
3. 线性回归: log N(ε) vs log(1/ε)
4. 返回斜率: d_H = -d(log N)/d(log ε)
```

**复杂度分析**:
- 时间复杂度: O(N × n_boxes)，其中N是点数
- 空间复杂度: O(N)
- 对于N > 100,000使用采样近似

#### 2.1.2 验证方法

**假设A验证公式**:

$$\dim_H(\Lambda) = 1 + \frac{1}{\log \text{Vol}(M)} \cdot \frac{L'(s_c)}{L(s_c)}$$

**验证步骤**:
1. 使用SnapPy计算双曲体积 Vol(M)
2. 通过文献或估计获取L-函数对数导数 L'/L
3. 计算预测维数并与数值估计比较
4. 统计分析相关性

#### 2.1.3 误差来源

| 误差来源 | 量级估计 | 控制方法 |
|----------|----------|----------|
| 盒计数数值误差 | ±0.05-0.10 | 增加采样点，多尺度拟合 |
| L-函数估计误差 | ±0.15-0.30 | 使用SageMath精确计算 |
| 体积计算误差 | < 0.01 | SnapPy高精度算法 |
| 系统偏差 | +0.61 (平均) | 群类型特定修正 |

### 2.2 p-adic多项式计算

#### 2.2.1 算法描述

**核心算法**: p-adic Bowen公式验证

```
算法: PAdicBowenVerification
输入: 素数 p, 度数 d
输出: Bowen维数 δ, 数值维数 dim_H

1. 计算p-adic赋值: v_p = v_p(d)
2. 若 v_p = 0:
   返回 "Bowen方程无有限解" (等距情形)
3. 计算Bowen维数:
   δ = log(d) / (v_p × log(p))
4. 数值计算Julia集维数 (盒计数)
5. 比较 δ 与 dim_H
```

#### 2.2.2 数值稳定性

**压力函数计算**:
$$P(s) = \log d - s \cdot v_p(d) \cdot \log p$$

稳定性分析:
- 对于大p: 使用对数空间计算避免溢出
- 对于纯p幂: δ = 1 精确成立 (代数恒等式)
- 数值误差: < 10⁻¹² (双精度浮点)

#### 2.2.3 收敛标准

| 情形 | 条件 | 收敛行为 |
|------|------|----------|
| 纯p幂 | d = p^k | 精确解 δ = 1 |
| 混合幂 | d = p^k × m | δ > 1 (需要修正) |
| 等距 | p ∤ d | 无有限解 |

### 2.3 统计分析

#### 2.3.1 交叉验证方法

**5折交叉验证**:
1. 将数据随机分为5个子集
2. 每次使用4个子集训练，1个子集测试
3. 计算每折的RMSE
4. 评估模型稳定性

```
Fold 1: RMSE = 0.561

Fold 3: RMSE = 0.904
Fold 4: RMSE = 0.679
Fold 5: RMSE = 0.977

```

#### 2.3.2 置信区间计算

**Pearson相关系数的95%置信区间**:

使用Fisher z变换:
$$z = \frac{1}{2}\ln\left(\frac{1+r}{1-r}\right)$$

置信区间:
$$z \pm \frac{1.96}{\sqrt{n-3}}$$

对于 r = 0.721, n = 15:
- z = 0.908
- 标准误 = 0.289
- 95% CI for z: [0.342, 1.474]
- **95% CI for r: [0.327, 0.896]**

#### 2.3.3 假设检验

**零假设 H₀**: 假设A与实际维数无相关性 (ρ = 0)

**备择假设 H₁**: 存在正相关 (ρ > 0)

检验结果:
- Pearson r = 0.721
- t统计量 = 3.78
- **p值 = 0.002 < 0.01**

**结论**: 拒绝H₀，假设A与实际维数存在显著正相关。

#### 4.2.3 收敛性

**纯p幂情形的数学证明**:

对于 $f(z) = z^{p^k}$:
$$\delta = \frac{\log d}{v_p(d) \cdot \log p} = \frac{\log(p^k)}{k \cdot \log p} = \frac{k \cdot \log p}{k \cdot \log p} = 1$$

由于 $\dim_H(J(f)) = 1$，我们有:
$$\dim_H(J(f)) = \delta = 1 \quad \checkmark$$

**验证**:
$$P(-\delta \cdot \log |f'|_p) = \log(p^k) - 1 \cdot k \cdot \log p = 0 \quad \checkmark$$
