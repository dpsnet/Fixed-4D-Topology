### ✅ P2.2: 非双曲Bowen公式反例

**交付物**: `proofs/NON_HYPERBOLIC_COUNTEREXAMPLE.md` (7页)

**反例构造**:
```
多项式: φ(z) = z² - 1/4 over ℚₚ (p ≥ 3)

性质:
- 中性不动点: z₀ = 1/2, |φ'(z₀)|ₚ = 1
- Julia集维数: dim_H(J(φ)) = 1
- 压力方程解: s* ≈ 0.73 < 1

结论: dim_H ≠ s*，Bowen公式失效
```

**失效机制**:
1. 中性点附近的非扩张动力学
2. "花状"结构的维数贡献
3. 压力函数无法捕捉非双曲部分

**定理5.1**: 抛物型p进多项式一般不满足Bowen公式
