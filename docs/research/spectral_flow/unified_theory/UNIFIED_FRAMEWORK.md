## 统一数学框架

### 1. 热核统一描述

所有系统共享热核渐近展开：

```
Θ(t) = c₄t⁻² + c₃t^(-3/2) + c₂t⁻¹ + c_δt^(-(1+δ)/2) + ...
       ↓      ↓       ↓         ↓
      4维    3维     2维      分形修正
```

### 2. 谱维度统一公式

```
d_s(ℓ) = -2 dlnΘ/dlnℓ = d_min + (d_max-d_min)/(1+(ℓ/ℓ₀)^(1/c₁))
```

## 理论意义

### 1. 维度是涌现性质

维度不是时空的基本属性，而是**约束强度**的涌现结果：
- 无约束 → 高维度 (d_max = 4)
- 强约束 → 低维度 (d_min = 2)

### 2. 全息原理的普适性

维度流支持全息原理跨越所有尺度：
- 黑洞视界: 最著名的全息系统
- 旋转系统: 经典全息
- 量子引力: 普朗克尺度全息

### 3. 信息论基础

c₁ 公式 c₁ = 1/2^(d-2+w) 来源于信息密度：
- c₁ < 1: 高信息密度，稳定中间维度
- c₁ = 1: 临界信息密度，2D的特殊性
- c₁ > 1: 低信息密度，快速维度流
