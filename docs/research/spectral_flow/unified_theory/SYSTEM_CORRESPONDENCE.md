### 各系统的约束机制

**1. 旋转系统 (E-6实验)**

```
离心势: V_cf = -½mω²r²

效应:
• 将粒子推向赤道平面
• 径向自由度被约束
• z方向运动被抑制
• 有效维度: 4 → 3 (z压缩) → 2.5 (径向约束)
```

**2. 黑洞系统 (史瓦西时空)**

```
引力势: Φ = -GM/r = -c²r_s/(2r)

效应:
• 物质被束缚在薄壳层
• 时间维度"冻结" (引力红移 → g_tt → 0)
• 径向自由度消失
• 有效维度: 4 → 3 (时间冻结) → 2 (视界)
```

**3. 量子引力 (普朗克尺度)**

```
量子涨落: ΔE · Δt ≥ ℏ/2

效应:
• 时空几何量子化
• 时空泡沫结构
• 连续流形破缺
• 有效维度: 4 → 3 → 2 (普朗克尺度)
```

    params : system-specific parameters
    """

        epsilon = params['omega']**2 * params['r']**2 / c**2
    elif system_type == 'blackhole':
        epsilon = params['r_s'] / params['r']
    elif system_type == 'quantum':
        epsilon = params['E'] / PLANCK_ENERGY

    d_eff = d_min + (d_max - d_min) / (1 + (epsilon/epsilon_c)**alpha)

## 理论意义

### 1. 经典-量子的桥梁

E-6实验揭示了经典系统和量子系统共享相同的**维度流动物理**。

### 2. 全息原理的普适性

维度流支持全息原理跨越所有尺度：
- 不仅是黑洞视界
- 任何强约束系统都展现类似行为

### 3. 维度不是基本的

维度是**约束强度的涌现性质**，而非时空的基本属性。
