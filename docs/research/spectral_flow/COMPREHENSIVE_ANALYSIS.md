## 2. 宏观物理解释

### 2.1 核心机制：约束减少自由度

**物理图像**:

```
无约束 → 弱约束 → 强约束 → 极强约束
  ↓        ↓        ↓         ↓
4维      3.5维     3维       2.5维
```

**机制解释**:

1. **旋转系统**
   - 离心力: F_cf = mω²r
   - 将粒子推向赤道平面
   - 径向自由度被约束
   - 有效维度降低

2. **黑洞系统**
   - 引力: F_g = GMm/r²
   - 物质被束缚在薄壳层
   - 时间维度"冻结"（引力红移）
   - 接近视界时d_eff → 2

3. **量子引力**
   - 量子涨落约束时空几何
   - 普朗克尺度下时空泡沫
   - UV极限下d_s → 2

### 2.2 数学模型

**统一维度流动公式**:

$$d_{eff}(\epsilon) = d_\infty + \frac{d_0 - d_\infty}{1 + (\epsilon/\epsilon_c)^\alpha}$$

**参数** (来自E-6实验拟合):
- $d_0 = 4.004$ (无约束)
- $d_\infty = 2.500$ (极强约束)
- $\epsilon_c = 0.896$ (特征约束)
- $\alpha = 1.698$ (过渡指数)

**拟合质量**: R² = 0.998
