## 核心发现

### 1. Giant Rydberg Excitons in Cu2O (Nature 2014)

**关键数据**:
- 观测到主量子数高达 **n = 25** 的Rydberg激子
- 这是当时已知最大的激子玻尔半径 (~数微米)
- 能级遵循修正的Rydberg公式

**物理参数**:
- 激子玻尔半径: a_B ≈ 1.1 nm
- Rydberg能量: Ry* ≈ 92 meV
- 介电常数: ε ≈ 7.5

**与我们的研究的相关性**: 
✅ **极高** - 提供高n态的精确光谱数据

### 2. Quantum Confined Rydberg Excitons (J. Phys. B 2020)

**关键内容**:
- 研究量子阱中受限的Rydberg激子
- 阱宽变化导致能级位移
- 高n态对维度效应更敏感

**实验设置**:
- Cu2O薄膜厚度: 10 nm - 1 μm
- 温度: < 10 K
- 光谱分辨率: 微eV级别

**与我们的研究的相关性**:
✅ **极高** - 直接研究维度受限效应

### 3. Rydberg Excitons at Millikelvin Temperatures (arXiv 2025)

**最新进展**:
- 毫开尔文温度下的高分辨率光谱
- 线宽极窄 (~MHz级别)
- 能级位置精确测量

**与我们的研究的相关性**:
✅ **高** - 提供提取c₁所需的精度

## 可检验的预测

基于我们的c₁ = 0.5理论，对Cu2O系统预测：

### 预测1: 量子亏损的能量依赖性

在3D→2D过渡区域（薄膜样品）：
```
δ(n) = 0.5 × (3 - d_eff(n))
     = 0.5 / (1 + (n/n₀)²)
```

**可观测效应**:
- 低n态: δ ≈ 0 (接近3D)
- 高n态: δ → 0.5 (接近2D)
- 过渡区(n ≈ n₀): δ ≈ 0.25

### 预测2: 能级间距的非Rydberg行为

标准Rydberg: ΔE_n ∝ 1/n³

维度流修正:
```
ΔE_n = Ry* × [1/(n-δ)² - 1/(n+1-δ')²]
```

其中δ, δ'随n变化。

### 预测3: 薄膜厚度与过渡量子数的关系

```
n₀ ∝ L/a_B
```

更薄的膜 → 更小的n₀ → 更快的维度流

## 备注

Cu2O系统相比GaAs量子阱的优势：
- ✅ 更高的量子数可达（n~25 vs n~5-10）
- ✅ 更大的玻尔半径（对维度效应更敏感）
- ✅ 已有大量 published 数据
- ⚠️ 但：激子-声子耦合较强

**建议优先级**: Cu2O > GaAs（数据可用性角度）
