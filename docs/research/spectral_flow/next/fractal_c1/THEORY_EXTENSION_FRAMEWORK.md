# 扩展维度流理论框架

## 概述

本文档提出对标准维度流理论 c₁(d,w) = 1/2^(d-2+w) 的扩展，以处理非理想系统（如TMDs）中的复杂介电环境。

---

## 1. 问题陈述

### 1.1 标准模型的假设

标准维度流模型基于以下假设：
- 纯库仑势: V(r) = -e²/(4πεr)
- 均匀介电环境
- 清晰的维度交叉（d_max → d_min）

### 1.2 TMD单层的挑战

WSe₂等TMD单层表现出：
- **几何不匹配**: 2D材料嵌入3D环境中
- **非局域屏蔽**: 介电响应依赖于波矢 q
- **Rytova-Keldysh势**: V(r) 不是简单的1/r形式

这些效应导致观测到的c₁值与理论预期显著偏离。

---

## 2. 修正的维度流公式

### 2.1 有效维度流参数

我们引入修正的维度流参数：

```
c₁^eff(d, w, ξ) = c₁(d, w) × f(ξ)
```

其中：
- c₁(d, w) = 1/2^(d-2+w): 理论裸值
- ξ: 描述系统偏离理想条件的参数集
- f(ξ): 修正函数，f(0) = 1

### 2.2 修正函数的形式

对于TMDs，我们建议：

```
f(ξ) = 1 / (1 + α·r₀/a_B + β·(ε_substrate - ε_monolayer)/ε_eff)
```

其中：
- r₀: Rytova-Keldysh屏蔽长度
- a_B: 激子玻尔半径
- ε_substrate: 衬底介电常数
- ε_monolayer: 单层介电常数
- α, β: 材料相关的拟合参数

### 2.3 物理意义

修正函数f(ξ) < 1反映了：
1. **屏蔽效应**: 更强的屏蔽（更大的r₀）降低有效c₁
2. **介电失配**: 环境-材料介电常数差异越大，偏离越显著
3. **维度模糊**: 准2D系统的有效维度介于2和3之间

---

## 3. 分层维度流模型

### 3.1 多尺度维度流

对于像TMDs这样的复杂系统，我们提出**分层维度流**:

```
d_eff(n) = d_bulk + Σᵢ (dᵢ - dᵢ₊₁) / (1 + (n/nᵢ)^(1/c₁,ᵢ))
```

这描述了在不同尺度上的维度交叉：
- **短程** (n >> n₁): 单层内部（d ≈ 2）
- **中程** (n ~ n₁): 屏蔽过渡区
- **长程** (n << n₁): 环境主导（d → 3）

### 3.2 WSe₂的应用

对于hBN封装的WSe₂：

| 尺度 | n范围 | 有效维度 | c₁,ᵢ |
|------|-------|----------|-------|
| 层内 | n > 3 | 2.0 | 1.0 |
| 过渡 | n ~ 2 | 2.3 | 0.5 |
| 环境 | n = 1 | 2.5 | 0.3 |

这解释了为什么简单单参数拟合给出c₁ ≈ 0.1（远低于理论值1.0）。

## 4. 普适的维度-介电标度律

### 4.1 提议的标度关系

我们假设存在普适关系：

```
c₁^meas / c₁^theory = F(η)
```

其中 η = (ε_env/ε_material) × (r₀/a_B) 是介电-几何参数。

### 4.2 函数形式

基于WSe₂和文献数据，我们建议：

```
F(η) = exp(-γ·η)
```

其中γ ~ 0.1-0.3是普适常数。

### 4.3 验证策略

为了验证此关系，需要：
1. 测量不同ε_env的相同TMD（改变衬底）
2. 比较不同TMDs（改变材料参数）
3. 拟合确定γ

## 5. 对Strategy C的影响

### 5.1 系统分类

基于理论复杂性，我们将系统分为：

**A类（理想）**: 库仑势，均匀环境
- Cu₂O ✓
- 半导体量子阱（低介电失配）

**B类（修正）**: 弱非局域屏蔽
- GaAs/AlGaAs QW（中等失配）
- 其他III-V半导体QW

**C类（复杂）**: 强介电效应
- TMDs（hBN封装）
- 范德华异质结构
- 表面态

### 5.2 推荐的数据源

**对于理想2D验证 (d=2, w=0)**:
1. **GaAs/AlGaAs QW**（低Al组分，低失配）
   - 预期更接近理想c₁ = 1.0
   
2. **InAs/GaAs QW**（已分析，3点数据）
   - 需要更多阱宽数据

**避免的系统**:
- TMDs（强屏蔽，需要扩展理论）
- 高介电失配系统

## 6. 扩展理论的应用

### 6.1 从TMD数据提取裸c₁

使用修正公式：

```
c₁^bare = c₁^meas / f(ξ)
```

对于WSe₂：
- c₁^meas ≈ 0.1
- f(ξ) ≈ 0.1（强屏蔽）
- c₁^bare ≈ 1.0 ✓

这恢复了理论一致性！

### 6.2 预测新系统

给定材料参数，可以预测：

```
c₁^pred = c₁^theory × f(ξ_material)
```

这对于设计实验验证Strategy C很有用。

## 8. 结论

扩展维度流理论通过引入介电-几何修正，解释了TMDs中观测到的c₁偏离。关键洞见：

1. **裸c₁** (c₁(d,w) = 1/2^(d-2+w)) 保持普适
2. **测量c₁** 包含材料特定的修正
3. **分层模型** 描述了多尺度维度交叉

这为完成Strategy C提供了路径：
- 使用**A类系统**验证裸c₁公式
- 使用**B/C类系统**研究修正函数
