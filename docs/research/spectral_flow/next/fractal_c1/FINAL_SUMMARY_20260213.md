## 关键公式汇总

### 维度流
```
d_eff(n) = 2 + 1 / (1 + (n/n₀)^(1/c₁))
```

### 量子亏损
```
δ(n) = 0.5 × (3 - d_eff)
     = 0.5 / (1 + (n₀/n)^(1/c₁))
```

### 能级公式
```
E_n = -Ry / (n - δ(n))²
```

### c₁理论预测
```
c₁(3) = 1/2^(3-2) = 0.5
c₁(4) = 1/2^(4-2) = 0.25
c₁(5) = 1/2^(5-2) = 0.125
```
