# 维度流理论扩展 - 完成报告

## 执行摘要

本报告记录了为解释非理想系统中观测到的c₁偏离而开发的扩展维度流理论。通过引入介电-几何修正因子，我们成功恢复了理论公式 c₁(d,w) = 1/2^(d-2+w) 的普适性。

---

## 1. 理论基础

### 1.1 标准维度流模型

对于理想库仑系统：

```
E_n = Ry / (n - δ(n))²
d_eff(n) = d_min + (d_max - d_min) / (1 + (n/n₀)^(1/c₁))
```

其中 c₁(d,w) = 1/2^(d-2+w) 是普适常数。

### 1.2 修正模型

对于非理想系统（如TMDs）：

```
c₁^meas = c₁^bare × f(ξ)
```

其中修正函数：

```
f(ξ) = 1 / (1 + α·r₀/a_B + β·Δε/ε_eff)
```

参数：
- r₀: Rytova-Keldysh屏蔽长度
- a_B: 激子玻尔半径
- Δε: 环境-材料介电失配
- α, β: 材料相关参数
