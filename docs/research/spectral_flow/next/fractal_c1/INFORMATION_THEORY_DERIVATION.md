## 🔬 推导 $c_1 = 1/2^{d-2}$

### 步骤1: 定义"几何信息密度"

考虑从d_max维度到d_min=2维的过渡。

在尺度ℓ处，有效维度为 $d_s(\ell)$。

**信息密度**（每单位尺度的信息）:
```
ρ_info(ℓ) ∝ d_s(ℓ) / ℓ
```

### 步骤2: 信息守恒

假设在谱流过程中，总信息守恒:
```
∫ ρ_info(ℓ) dℓ = const
```

### 步骤3: 指数稀释的解释

**关键洞察**:
- 每增加一维，信息需要"分配"到新的方向
- 如果信息均匀分布，每维获得 1/2 的信息份额
- 这导致指数衰减: $2^{-(d-4)}$

**数学表述**:

设4维时的信息为 $I_0$。

在d维中，信息被分配到 (d-2) 个"活跃"维度（从2到d）。

如果每次维度增加都使信息减半:
```
I_d = I_4 × (1/2)^{d-4}
```

这与c1的行为一致！

### 步骤4: 与c1联系

回顾谱维度流公式:
```
d_s(ℓ) = d_min + (d_max - d_min) × f(ℓ/c1)
```

c1控制过渡的"陡峭程度"。

**解释**: c1反映了信息的"集中程度"
- c1大 → 信息集中 → 过渡陡峭
- c1小 → 信息分散 → 过渡平缓

在d维中:
- 高维(d>4): 信息分散，c1小，过渡平缓
- 低维(d<4): 信息集中，c1大，过渡陡峭

**定量关系**:
```
c1(d) = c1(4) × 2^{-(d-4)} = (1/4) × 2^{-(d-4)} = 1/2^{d-2}
```
