## 🧮 数学重写

方案5可以写成:
```
c_1 = 1 / 2^{d-2} = 2^{-(d-2)} = 4 * 2^{-d} = 4 / 2^d
```

或者:
```
c_1 = (1/2)^{d-2} = (1/4) * (1/2)^{d-4}
```

以4维为基准:
```
c_1(d) = c_1(4) * (1/2)^{d-4} = 0.25 * 2^{-(d-4)}
```
