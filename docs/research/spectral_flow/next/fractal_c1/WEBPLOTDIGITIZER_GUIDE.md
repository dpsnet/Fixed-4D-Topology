# WebPlotDigitizer 使用指南

## 目标
从PDF论文图表中提取数值数据

---

## 工具访问

**网址**：https://apps.automeris.io/wpd/

**特点**：
- 免费在线工具
- 无需安装
- 支持多种图表类型

---

## 操作步骤

### 1. 准备截图
从PDF论文中截取清晰的图表：
- 包含完整坐标轴
- 数据点清晰可见
- 图例完整

### 2. 上传图片
- 点击 "Load Image"
- 选择截图文件
- 或拖拽图片到页面

### 3. 选择图表类型
- **2D (X-Y) Plot**：标准散点图/线图
- **Bar Chart**：柱状图
- **Polar Diagram**：极坐标图
- **Ternary Diagram**：三元相图

**选择**：2D (X-Y) Plot

### 4. 校准坐标轴
**X轴校准**：
- 点击"Align Axes"
- 选择"X Axis"
- 点击X轴左端点 → 输入数值
- 点击X轴右端点 → 输入数值

**Y轴校准**：
- 选择"Y Axis"
- 点击Y轴下端点 → 输入数值
- 点击Y轴上端点 → 输入数值

**对数坐标**：勾选"Log Scale"（如需要）

### 5. 提取数据点
**手动模式**：
- 点击"Acquire Data"
- 逐个点击数据点
- 工具自动记录坐标

**自动模式**（如果数据点清晰）：
- 点击"Automatic Extraction"
- 选择颜色/算法
- 自动识别数据点

### 6. 导出数据
- 点击"View Data"
- 选择导出格式：CSV / JSON / MATLAB
- 下载文件

## 针对本项目的数据提取

### 目标图表1：石墨烯Landau能级
**X轴**：磁場 B (T) 或 Landau index n  
**Y轴**：能量 E (meV) 或跃迁能量

**关键数据**：
- n = 0, ±1, ±2, ±3... 的能级位置
- 不同磁场下的能量

### 目标图表2：量子阱能级vs厚度
**X轴**：阱宽 L (nm) 或 n (主量子数)  
**Y轴**：能量 E (meV) 或束缚能

**关键数据**：
- 不同阱宽（5nm, 10nm, 20nm, 50nm）
- n = 1, 2, 3... 的能级

## 注意事项

### 提高精度
- 使用高分辨率PDF（300 dpi+）
- 截图时放大图表
- 多次点击同一数据点取平均

### 误差估计
- 记录数据点的大小（FWHM）
- 作为能量不确定度
- 典型值：0.1-1 meV

### 数据验证
- 检查提取的数据是否合理
- 与论文中报告的值比较
- 绘制快速图表验证趋势
