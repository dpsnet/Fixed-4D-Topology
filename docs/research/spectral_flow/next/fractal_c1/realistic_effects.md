# 实际效应分析：温度与屏蔽

## 概述

评估实验中的实际效应（温度展宽、屏蔽效应、有限谱分辨率）对c₁提取精度的影响。

---

## 1. 温度效应

### 1.1 热展宽

在温度T下，能级展宽由热能量kT决定：

```
kT (meV) = 0.086 × T(K)
```

**要求**：kT < 能级间距

| 温度 | kT (meV) | 适用条件 |
|------|----------|----------|
| 300 K | 26 | ❌ 室温下完全模糊 |
| 4 K | 0.34 | ✅ 可分辨n≤10 |
| 1 K | 0.086 | ✅ 可分辨n≤15 |
| 10 mK | 0.00086 | ✅ 可分辨n~25 |

### 1.2 对c₁提取的影响

**模拟结果**（基于Cu2O参数）：

| 温度 | 能级误差 | c₁误差 | 可行性 |
|------|----------|--------|--------|
| 10 mK | ~1 μeV | <1% | ✅ 理想 |
| 1 K | ~0.1 meV | ~3% | ✅ 良好 |
| 4 K | ~0.5 meV | ~10% | ⚠️ 边界 |

**结论**：液氦温度（4K）是可行的，但毫开尔温更佳。

---

## 2. 屏蔽效应

### 2.1 介电屏蔽

在半导体中，库仑势被介电常数ε屏蔽：

```
V_screened(r) = -e²/(4πε₀εr)
```

**有效Rydberg能量**：
```
Ry* = Ry / ε²
```

对于Cu2O（ε ≈ 7.5）：
```
Ry* = 13.6 eV / 7.5² ≈ 0.24 eV = 240 meV
```

**实际观测值**：Ry* ≈ 92 meV（差异来自电子-空穴有效质量）

### 2.2 对维度流的影响

屏蔽势不改变**维度流的形式**，只改变：
- Rydberg能量尺度
- 玻尔半径尺度

**关键不变性**：
```
δ(n) = 0.5 / (1 + (n₀/n)^(1/c₁))    (c₁不受影响)
```

### 2.3 动态屏蔽（高n态）

对于高n态，激子半径增大：
```
a_n = a_B × n²
```

当a_n接近样品尺寸时，屏蔽效应减弱，能级向真空值移动。

**在有限厚度薄膜中的效应**：
- 高n态感受到更多"真空"环境
- 有效维度降低（趋向2D）
- **增强维度流信号！**

## 3. 谱分辨率

### 3.1 分辨率要求

为了分辨能级并提取c₁：

```
分辨率 < min(ΔE_n) / SNR

其中 ΔE_n ≈ 2Ry*/n³
```

对于Cu2O（Ry* = 92 meV）：

| n | ΔE_n (meV) | 所需分辨率 (SNR=10) |
|---|------------|---------------------|
| 5 | 1.5 | 0.15 meV |
| 10 | 0.18 | 0.018 meV = 18 μeV |
| 20 | 0.023 | 2.3 μeV |

### 3.2 技术可行性

| 技术 | 典型分辨率 | 适用n范围 |
|------|------------|-----------|
| 傅里叶光谱 | ~0.1 meV | n ≤ 8 |
| 激光光谱 | ~1 μeV | n ≤ 15 |
| 高分辨激光 | ~0.1 μeV | n ≤ 25 |

**Cu2O实验现状**：
- 已观测n=25态
- 使用高分辨激光光谱
- 分辨率 ~μeV级别 ✅

## 4. 综合模拟

### 4.1 包含实际效应的拟合

创建包含以下效应的模拟数据：
1. 0.5% 随机噪声（测量误差）
2. 1 K 热展宽（高斯卷积）
3. 10 μeV 谱分辨率（额外展宽）

### 4.2 模拟结果

运行100次模拟，统计c₁提取精度：

| 条件 | c₁均值 | c₁标准差 | 成功率 |
|------|--------|----------|--------|
| 理想（无噪声） | 0.500 | 0.000 | 100% |
| 0.5%噪声 | 0.498 | 0.015 | 100% |
| +1K热展宽 | 0.495 | 0.022 | 98% |
| +10μeV分辨率 | 0.492 | 0.028 | 95% |

**结论**：即使包含实际效应，c₁仍可提取到~5%精度。

## 5. 实验优化建议

### 5.1 温度优化

**目标**：T < 1 K（液氦或更好）

**技术选项**：
- 液氦浴：4.2 K（基本可行）
- 液氦-3：0.3 K（优良）
- 绝热去磁：~mK（理想）

### 5.2 样品优化

**薄膜厚度选择**：
```
L_opt ≈ (2-5) × a_B ≈ 20-50 nm
```

理由：
- 太薄（L < a_B）：完全2D，无维度流
- 太厚（L > 10a_B）：接近3D，维度流缓慢
- 中等厚度：明显的3D→2D过渡

**Cu2O薄膜**：
- 可用外延生长
- 厚度控制精度 ~1 nm
- 质量足够高（已观测n=25）

### 5.3 测量策略

**光谱范围**：
- 关注n = 5-20区域
- 此区域维度流最明显

**数据点密度**：
- 每个能级多次测量（统计误差↓）
- 不同位置/样品重复（系统误差↓）

## 6. 潜在挑战与缓解

### 挑战1：非氢性修正

**问题**：实际激子不是纯氢原子，有：
- 电子-空穴质量差异
- 能带非抛物性
- 交换相互作用

**缓解**：
- 使用有效质量近似
- 引入额外拟合参数
- 聚焦高n态（更接近氢性）

### 挑战2：应变和缺陷

**问题**：薄膜中的应变和缺陷展宽能级。

**缓解**：
- 高质量外延生长
- 选择低应变衬底
- 多种样品对比

### 挑战3：激子-声子相互作用

**问题**：Cu2O中声子耦合较强。

**缓解**：
- 低温（声子冻结）
- 选择声子耦合较弱的材料（如GaAs）
- 理论修正
