## 观察与结论

### 主要发现

1. **Julia集结构**:
   - 对于 f(z) = z^d，Julia集是p-adic单位圆，维数为1
   - 添加低阶项会改变Julia集的精细结构
   - 在p-adic情形下，Julia集表现出与复数情形不同的分形特性

2. **Bowen方程适用性**:
   - 对于纯幂映射 f(z) = z^d，当 p|d 时Bowen方程严格成立
   - 对于一般多项式，Bowen方程的解可能与实际维数有偏差
   - 偏差大小与低阶项的p-adic范数相关

3. **数值结果模式**:
   - 大部分测试的数值维数接近1
   - 扰动较小时，Bowen δ 与维数的误差较小
   - 线性项系数被p整除时，对维数影响较小

### 理论猜想

基于数值探索，我们提出以下猜想：

**猜想1 (扰动稳定性)**: 对于p-adic多项式 f(z) = z^d + g(z)，其中 deg(g) < d，
如果 ||g||_p 足够小，则
```
dim_H(J(f)) = δ + O(||g||_p)
```
其中 δ 是Bowen方程的解。

**猜想2 (扩张条件)**: 对于扩张型p-adic多项式（即 |f'(z)|_p > 1 在Julia集上），
存在修正的Bowen公式：
```
dim_H(J(f)) = inf{t : P(-t·log|f'|_p) ≤ 0}
```

**猜想3 (临界指数)**: 存在一个临界指数 δ_c，使得当扰动的p-adic范数
小于 δ_c 时，Julia集的拓扑结构保持稳定。
