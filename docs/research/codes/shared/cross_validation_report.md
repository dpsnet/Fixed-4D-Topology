### 1.2 误差分析体系

#### 1.2.1 系统误差估计

系统误差通过以下方法估计：

**偏差估计公式（Jackknife）：**

$$\hat{b}_{jack} = (n-1)(\hat{\theta}_{(\cdot)} - \hat{\theta})$$

其中：
- $\hat{\theta}$: 完整样本估计值
- $\hat{\theta}_{(\cdot)}$: 留一法估计的平均值
- $n$: 样本量

**相对偏差：**

$$\text{相对偏差} = \frac{\hat{b}}{\theta_{\text{true}}} \times 100\%$$

#### 1.2.2 随机误差估计

| 指标 | 公式 | 说明 |
|------|------|------|
| 标准差 (SD) | $\sigma = \sqrt{\frac{1}{n-1}\sum(x_i - \bar{x})^2}$ | 离散程度 |
| 标准误差 (SEM) | $\text{SEM} = \frac{\sigma}{\sqrt{n}}$ | 均值精度 |
| 变异系数 (CV) | $\text{CV} = \frac{\sigma}{\bar{x}} \times 100\%$ | 相对变异 |
| MAD | $\text{MAD} = \text{median}(|x_i - \text{median}(x)|)$ | 稳健离散度 |

#### 1.2.3 误差传播

对于函数 $y = f(x_1, x_2, ..., x_n)$，合成不确定度：

$$u_c(y) = \sqrt{\sum_{i=1}^n \left(\frac{\partial f}{\partial x_i}\right)^2 u^2(x_i) + 2\sum_{i<j}\frac{\partial f}{\partial x_i}\frac{\partial f}{\partial x_j}u(x_i)u(x_j)r(x_i,x_j)}$$

### 1.3 Bootstrap方法

#### 1.3.1 百分位置信区间

对于置信水平 $1-\alpha$：

$$CI_{percentile} = [\theta^*_{(\alpha/2)}, \theta^*_{(1-\alpha/2)}]$$

其中 $\theta^*_{(p)}$ 是Bootstrap分布的第 $p$ 百分位数。

#### 1.3.2 BCa置信区间

Bias-Corrected and Accelerated (BCa) 区间：

$$CI_{BCa} = [\theta^*_{(\alpha_1)}, \theta^*_{(\alpha_2)}]$$

其中：
- $\alpha_1 = \Phi(z_0 + \frac{z_0 + z_{(\alpha/2)}}{1 - a(z_0 + z_{(\alpha/2)})})$
- $\alpha_2 = \Phi(z_0 + \frac{z_0 + z_{(1-\alpha/2)}}{1 - a(z_0 + z_{(1-\alpha/2)})})$
- $z_0 = \Phi^{-1}(\hat{F}(\hat{\theta}))$: 偏差校正
- $a$: 加速度参数（通过Jackknife估计）

### 1.4 统计检验

#### 1.4.1 正态性检验

| 检验名称 | 统计量 | 适用场景 |
|----------|--------|----------|
| Shapiro-Wilk | $W$ | 小样本 ($n \leq 5000$) |
| D'Agostino-Pearson | $K^2$ | 大样本 |
| Anderson-Darling | $A^2$ | 分布拟合 |
| Jarque-Bera | $JB$ | 偏度峰度 |

#### 1.4.2 异方差性检验

**Breusch-Pagan检验：**

$$LM = n \cdot R^2_{\hat{u}^2 \sim \hat{y}}$$

其中 $R^2$ 是残差平方对拟合值回归的决定系数。

#### 1.4.3 模型选择准则

| 准则 | 公式 | 特点 |
|------|------|------|
| AIC | $2k - 2\ln(\hat{L})$ | 平衡拟合与复杂度 |
| AICc | $AIC + \frac{2k(k+1)}{n-k-1}$ | 小样本修正 |
| BIC | $k\ln(n) - 2\ln(\hat{L})$ | 更倾向简约模型 |

**AIC权重：**

$$w_i = \frac{\exp(-\Delta_i/2)}{\sum_j \exp(-\Delta_j/2)}$$

其中 $\Delta_i = AIC_i - AIC_{min}$

### 8.3 统计显著性论证

**假设检验框架：**

- **H0（零假设）：** 维数估计无意义（随机噪声）
- **H1（备择假设）：** 维数估计具有统计显著性

**检验统计量：**

$$t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}} = \frac{1.5229 - 1.0}{0.4225/\sqrt{42}} = 8.0234$$

**p值：** $p < 0.0001$

**结论：** 强烈拒绝H0，维数估计具有高度统计显著性。

### 8.4 效应量分析

**Cohen's d：**
